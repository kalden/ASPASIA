Detailed Description

Takes output from an SBML Solver and uses this to create a new SBML Model.

Also capable of then changing the value of some of these responses to introduce an intervention. Then outputs the new SBML Model file

Member Function Documentation

void modelCreation.SBMLFileFromSolverOutput.newModelFromSteadyState ( )

Creates a new SBML model file by reading the final line of the SBML solver output file. Creates a map of parameter to value such that the parameters that are being altered via an intervention can easily be found. These are then assigned values set in the settings file.
