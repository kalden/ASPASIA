Provides functions to read in and process information from XML files.

This can be used to read in simulation information and process the sensitivity analysis settings file

static boolean xmlFileUtilities.XMLFileUtilities.readParameterInfo ( String  analysisMethod)
static

Used in Sensitivity analysis experiments - reads in parameter information for the analysis. This is stored in the appropriate arrays: parameter name, type of parameter, and min, max, and increment values. The latter three are only required for robustness analysis and as such may not exist in LHC and eFAST analyses.
